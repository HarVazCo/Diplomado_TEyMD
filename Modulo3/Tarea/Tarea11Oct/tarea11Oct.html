Prueba de Chow

La prueba de Chow nos dice si los coeficientes de una regresión son los mismos para un conjunto de datos dividido. Considerando el modelo general con kkk variables explicativas y una constante, y una muestra con dos grupos: g=1,2g=1,2g=1,2.

Yg=β1,g+β2,gx2,g+⋯+βk,gxk,g+ug Y_{g} = \beta_{1,g} + \beta_{2,g}x_{2,g} + \dots + \beta_{k,g} x_{k,g} + u_g Yg​=β1,g​+β2,g​x2,g​+⋯+βk,g​xk,g​+ug​

La prueba de Chow busca probar si los coeficientes y las constantes son iguales en los dos, es decir,

H0:β1,1=β1,2,β2,1=β2,2,…,βk,1=βk,2,H_0 : \beta_{1,1} = \beta_{1,2}, \beta_{2,1} = \beta_{2,2}, \dots, \beta_{k,1} = \beta_{k,2},H0​:β1,1​=β1,2​,β2,1​=β2,2​,…,βk,1​=βk,2​,

esta hipótesis incluye k+1k+1k+1 restricciones.

La prueba consiste en realizar tres estimaciones con el modelo general, una para cada grupo (Y1Y_1Y1​ y Y2Y_2Y2​) y la tercera para el conjunto de datos completo (Y)(Y)(Y). Ya que se tienen las estimaciones, se calcula la suma de residuales cuadrados de cada uno de los modelos:

SRC1=∑u^12SRC2=∑u^22SRC=∑u^2.\begin{align*} SRC_1 &= \sum \hat{u}_1^2 \\ SRC_2 &= \sum \hat{u}_2^2 \\ SRC &= \sum \hat{u}^2 \end{align*}.SRC1​SRC2​SRC​=∑u^12​=∑u^22​=∑u^2​.

Ya que se tienen las sumas, se calcula el estadístico FFF de la siguiente manera:

F=(SRC−(SRC1+SRC2))k+1SRC1+SRC2(n1+n2−2(k+1)) F = \frac{\frac{(SRC - (SRC_1 + SRC_2))}{k+1} }{\frac{SRC_1 + SRC_2}{(n_1 + n_2 - 2(k+1))} } F=(n1​+n2​−2(k+1))SRC1​+SRC2​​k+1(SRC−(SRC1​+SRC2​))​​

donde n1n_1n1​ y n2n_2n2​ son el número de observaciones del grupo 111 y del grupo 222 respectivamente.

Si no hay un cambio estructural, es decir, Y1Y_1Y1​ y Y2Y_2Y2​ son las mismas, SCRSCRSCR y SCR1+SCR2SCR_1+SCR_2SCR1​+SCR2​ no deben ser estadísticamente diferentes. Por lo tanto:

F∼F[k+1,(n1+n2−2(k+1))]. F \sim F_{[k+1,(n_1+n_2-2(k+1))]}. F∼F[k+1,(n1​+n2​−2(k+1))]​.

De esta manera, cuando FFF no excede el valor crítico F[k+1,(n1+n2−2(k+1))] F_{[k+1,(n_1+n_2-2(k+1))]} F[k+1,(n1​+n2​−2(k+1))]​, no se rechaza la hipótesis nula de estabilidad paramétrica. Caso contrario cuando FFF es mayor que el valor crítico F[k+1,(n1+n2−2(k+1))] F_{[k+1,(n_1+n_2-2(k+1))]} F[k+1,(n1​+n2​−2(k+1))]​, se rechaza la hipótesis nula y se concluye que Y1Y_1Y1​ y Y2Y_2Y2​ son diferentes.

Observaciones de la prueba de Chow:

  • Se generaliza fácilmente para casos de más de un cambio estructural.
  • Dado que la prueba de Chow es simplemente una prueba F, sólo es válida bajo homocedasticidad. Primero se debe asegurar que las varianzas de los residuos de Y1Y_1Y1​ y Y2Y_2Y2​ son las mismas.
  • Solo dice si las regresiones son las mismas, pero no especifica si difieren en las constantes o en los coeficientes.
  • Supone que se conocen los puntos de cambio estructural.

Prueba RV de Quandt (QLR)

La prueba RV de Quandt permite comprobar la existencia de cambios estructurales cuando se desconoce la fecha de ruptura. QLR=max[F(τ0),F(τ0+1),…,F(τ1)]. QLR = \text{max}[F(\tau_0),F(\tau_0+1),\dots,F(\tau_1)].QLR=max[F(τ0​),F(τ0​+1),…,F(τ1​)].

Observaciones de la prueba RV de Quandt:

  • No se necesita conocer el punto de quiebre.
  • Es pesada computacionalmente, ya que requiere estimar el modelo para cada posible punto de quiebre.
